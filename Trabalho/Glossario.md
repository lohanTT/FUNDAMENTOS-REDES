# Glossário do trabalho.
**Aqui estarão as definições, explicações e apontamentos para os termos encontrados no trabalho prático da dsicplina**
***
## Hub
### Definição
Dispositivo de rede que atua na camada física do [modelo OSI](#). A principal função de um hub é simplesmente repetir os dados recebidos em todas as portas para que todos os dispositivos conectados à rede recebam esses dados.

Não são dispositivos inteligentes, pois não possuem capacidade de tomar decisões sobre para onde enviar os dados e as transmite por inundação. O uso de hubs pode levar a problemas de colisões de dados em [redes Ethernet](#rede-ethernet).

Devido às limitações dos hubs, eles foram amplamente substituído por [switches](#switch) em redes modernas.

## Switch
### Definição
Dispositivo de rede que atua na camada de enlace de dados do [modelo OSI](). A principal função de um switch é encaminhar o tráfego de rede com base no endereço MAC dos dispositivos conectados à rede.

Switches são capazes de criar tabelas de comutação (tabelas MAC), que associam endereços MAC às portas físicas as quais os dispositivos estão conectados. Isso permite que os switches tomem decisões de comutação de forma rápida e precisa.

Tipos de switches:
- Não gerenciável: Não é possível configurar ou monitorar individualmente as portas do switch. Eles simplesmente encaminham o tráfego com base no endereço MAC sem intervenção do usuário. São ideais para redes domésticas ou pequenas empresas onde a simplicidade e o baixo custo são mais importantes do que a personalização ou controle avançado.
- Gerenciável: Permite que os administradores de rede ajustem o comportamento dele. É possível configurar portas individualmente, definir políticas de segurança, monitorar o tráfego, realizar segmentação de rede, entre outras funcionalidades. Além disso, permitem a detecção de problemas na rede, monitoramento de tráfego e análise de desempenho.
- De camada 3: Possui funcionalidadde de roteamento IP. Isso significa que eles podem encaminhar tráfego com base em endereços IP, além de endereços MAC. São frequentemente usados para segmentar redes em sub-redes e rotear o tráfego entre elas, o que é útil para melhorar o desempenho e a segurança da rede.

## Rede Ethernet
### Definição
É um tipo de rede de computadores com fio que usa tecnologia Ethernet para conectar dispositivos em uma LAN ou em redes locais maiores.
- Meio físico com fio: As redes Ethernet tradicionais usam cabos de par trançado, fibra óptica ou cabo coaxial para conectar dispositivos. Isso proporciona uma conexão física estável e confiável.
- Protocolo Ethernet: A Ethernet opera com base em um conjunto de protocolos e padrões conhecidos coletivamente como Ethernet. O protocolo Ethernet define como os dispositivos na rede se comunicam, como os quadros de dados são formatados e como os endereço MAC são usados para encaminhar os dados.
- Comutação Ethernet: O uso de switches Ethernet permite uma comutação mais eficiente e inteligente de dados, minimizando colisões de pacotes e tornando a rede mais eficiente.
- Velocidade de transmissão: As redes Ethernet podem operar em várias velocidades, desde 10 Mbps até 1 Gbps, 10 Gbps, 40 Gbps e até 100 Gbps em redes mais recentes
